# Claude Code Web

**Используйте Claude Code прямо в браузере — без терминала.**

Лёгкий веб-интерфейс, оборачивающий [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code/overview) с поддержкой постоянных сессий, индикаторов использования в реальном времени, переключения моделей и вложений файлов. Работает на **macOS** и **Windows**.

![Claude Code Web Screenshot](docs/screenshot.png)

> **Session sidebar** (left) · **Diff view for edits** (center) · **S/W/M usage gauges** (top right)

---

## Зачем этот проект?

Claude Code CLI мощный инструмент, но требует работы в терминале. Этот проект оборачивает его в удобный браузерный интерфейс, доступный каждому члену команды.

---

## Возможности

### Интерфейс чата
- Потоковые ответы с рендерингом Markdown (блоки кода, таблицы, списки)
- Поддержка IME для азиатских языков
- Перенос строки через Shift+Enter

### Отображение инструментов
- **Edit** — diff-просмотр (удаления красным, добавления зелёным)
- **Bash** — стиль терминала (приглашение `$` + вывод)
- **Read** — путь к файлу + содержимое
- Сворачиваемая сводка при 4+ инструментах: _"90 tools used (Bash ×12, Read ×65…)"_

### Управление сессиями
- **Постоянное хранение** — в SQLite, сохраняется после перезапуска
- **Восстановление сессии** — возобновляет сессию Claude CLI через `--resume` (без лишних токенов)
- **Branch** — разветвление текущего разговора
- **Авто-название** — генерируется из первого сообщения
- **Переименование / удаление** прямо в списке

### Индикаторы использования (S / W / M)

| Индикатор | Значение |
|-----------|---------|
| **S** | Использование за 5 часов % + время до сброса |
| **W** | Недельное использование всех моделей % + день/время сброса |
| **M** | Недельное использование Sonnet % |

### Переключение модели

| Модель | Вход | Выход | Рекомендуется для |
|--------|------|-------|------------------|
| **sonnet** (по умолчанию) | $3/M | $15/M | Обычная разработка |
| **opus** | $15/M | $75/M | Сложный анализ |
| **haiku** | $0.8/M | $4/M | Быстрые правки |

### Режимы разрешений
| Режим | Поведение |
|-------|----------|
| `acceptEdits` (по умолчанию) | Все инструменты + редактирование файлов |
| `auto` | Все инструменты авто-одобрены |
| `plan` | Только чтение |

### Slash-команды
Введите `/` для автодополнения:
- `/clear` — сброс чата + новая сессия
- `/branch` — разветвить разговор
- `/help` — список команд

### Вложения файлов
- Кнопка скрепки, перетаскивание или вставка Ctrl/Cmd+V
- Предпросмотр: миниатюры изображений + иконки файлов

## Требования

- **Node.js** 18+
- **Claude Code CLI** установлен и аутентифицирован

```bash
npm install -g @anthropic-ai/claude-code
claude
```

---

## Установка

```bash
git clone https://github.com/sonwonkyu/ClaudeCliWindowsMac.git
cd ClaudeCliWindowsMac
npm install
```

---

## Активация индикаторов (Опционально)

**macOS / Linux** (требуется `jq` — `brew install jq`)
```bash
bash scripts/setup-statusline.sh
```

**Windows (PowerShell)**
```powershell
powershell -ExecutionPolicy Bypass -File scripts/setup-statusline.ps1
```

---

## Запуск

```bash
npm start
```

Откройте `http://localhost:3333` в браузере.

**Другой порт:**
```bash
PORT=8080 npm start
```

---

## Десктопное приложение (Electron)

`desktop/` содержит обёртку Electron с системным треем, автозапуском сервера и окном входа для опроса использования.

**macOS:**
```bash
cd desktop && npm install && bash build-app.sh
```

**Windows:**
```bash
cd desktop && npm install && npm run build:win
```

---

## Примечания

- `data.db` создаётся автоматически при первом запуске
- Модель по умолчанию: **sonnet**
- **Не открывайте этот сервер в интернет** — нет аутентификации. Только для локального использования.
